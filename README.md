## 📌 Overview

**Car Diagnostic & Repair System** is an intelligent, interactive web platform that enables users to diagnose vehicle problems with ease — no mechanical expertise required. Built on a rule-based inference engine, the system analyzes user-reported symptoms through a guided Yes/No interface and produces confidence-ranked diagnostic results with repair guidance.

Developed as a university capstone project and nominated for a university-level technology exhibition due to its real-world applicability and system design excellence.

## ✨ Key Features

- 🔍 **Intelligent Symptom Analysis** — Rule-based inference engine that maps user symptoms to probable vehicle faults
- 📊 **Confidence-Based Results** — Weighted reasoning ranks issues by likelihood, giving users prioritized repair paths
- 🖱️ **Simple Yes/No Interface** — Step-by-step interaction designed for users with no mechanical background
- ⚡ **Fast Diagnosis** — Reduces time and cost of vehicle maintenance by narrowing down issues quickly
- 📱 **Responsive Design** — Works seamlessly across desktop and mobile devices

---

## 🧠 How It Works

```
User reports symptoms
        ↓
Rule-based inference engine processes inputs
        ↓
Weighted reasoning ranks possible faults
        ↓
Confidence-based diagnostic results presented
        ↓
Repair guidance provided
```

The system replaces the guesswork of vehicle maintenance with a structured, logic-driven approach — bridging the gap between complex automotive mechanics and everyday drivers.

## 🚀 Getting Started

Since this is a web-based system hosted on GitHub Pages, no installation is required.

Simply visit the live demo:
```
https://amir-mulham.github.io/Cars-diagnosis-problems/
```

Or clone the repository to run locally:

```bash
git clone https://github.com/AMIR-MULHAM/Cars-diagnosis-problems.git
cd Cars-diagnosis-problems
# Open index.html in your browser
```
